# Postiz Pinterest Privacy Policy

Minimal static site for a privacy-policy page suitable for a self-hosted Postiz Pinterest integration.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Push this folder to the repository.
3. In the repository settings, enable GitHub Pages from the default branch root.
4. Use the resulting public URL as the privacy-policy URL in the Pinterest developer form.

## Before publishing

- Replace the placeholder email in `index.html`.
- If this is not personal/internal use, update the policy text to match your actual data handling.

## Local preview

Open `index.html` in a browser.
