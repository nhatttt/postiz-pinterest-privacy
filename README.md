# Postiz Pinterest Privacy Policy

Static GitHub Pages site for the nhatprints.etsy privacy policy used with a self-hosted Postiz Pinterest integration.

## Published URL

After GitHub Pages is enabled, use this company-branded URL in the Pinterest developer form:

- `https://nhatttt.github.io/postiz-pinterest-privacy/nhatprints.etsy/`

## Repo structure

- `nhatprints.etsy/index.html`: privacy policy page
- `index.html`: simple redirect from the site root to the nhatprints.etsy page
- `.github/workflows/deploy-pages.yml`: GitHub Pages deployment workflow

## Before publishing

- Confirm the contact email in `nhatprints.etsy/index.html` is correct.
- Enable GitHub Pages in the repository settings with `Source = GitHub Actions`.
